# Sequenz e.V. — Öffentliche Dokumentation

Dieses Repository enthält die **öffentlich zugänglichen Dokumente** von [Sequenz e.V.](https://sequenz-ev.de)

## Über Sequenz

Sequenz e.V. ist ein gemeinnütziger Verein, der Underground Techno-Events organisiert.

## Inhalt

Hier befinden sich u.a.:

- **Satzung, Beitragsordnung, Verhaltenskodex** — die Grundlagendokumente des Vereins
- **Event-Richtlinien** — allgemeine Checklisten und Konzepte
- **Brand-Assets** — Logos und Styleguide
- **Rechtliches** — Datenschutz, Impressum

## Automatisch synchronisiert

Dieses Repository wird automatisch aus dem internen Dokumentations-Repository gespiegelt. Es werden nur Dokumente veröffentlicht, die explizit als `visibility: public` markiert sind.

**Manuelle Änderungen in diesem Repository werden beim nächsten Sync überschrieben.**
